Project Name: World Surf

Project Description

Designed to help competitors register and to inform what they are getting themselves into when it comes to the world wide competitions. This website is designed to inform competitors about how difficult it may be to compete world wide. This is a testing location that users will use if they are looking for surfing locations. It is designed for competitiveness and for leisure surfing.
